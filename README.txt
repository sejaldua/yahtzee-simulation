All code can be found in yahtzee.ipynb. Additionally, some code is included in the Appendix of the report.

To run the code, you will need Python3. You will also need to install the following libraries:
- pandas: for dataframe manipulation
- numpy: for random sampling functions
- matplotlib: for visualization
- seaborn: for visualization
- scipy: for statistical testing
- tqdm: (optional) for being able to see when simulations will finish running
